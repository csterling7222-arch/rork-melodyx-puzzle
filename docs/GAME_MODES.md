# Melodyx Game Modes

## Overview

Melodyx offers multiple game modes to cater to different play styles and skill levels. Each mode provides a unique musical puzzle experience.

## Daily Challenge

The core gameplay mode where players guess a mystery melody note-by-note.

### Rules
- One new puzzle every day at midnight (local time)
- 6 attempts to guess the correct melody
- Melody length varies by difficulty (4-6 notes)
- Color-coded feedback after each guess:
  - **Green**: Correct note in correct position
  - **Yellow**: Correct note in wrong position
  - **Gray**: Note not in the melody

### Scoring
- Points based on number of guesses used
- Streak bonuses for consecutive daily wins
- Perfect solve bonus (first try)

## Sound Fever

An endless mode with increasing difficulty and special mechanics.

### Features
- Unlimited puzzles with progressive difficulty
- Combo system for consecutive correct answers
- Score multipliers
- Boss battles every 10 levels
- Power-ups and special items

### Boss Battles
- Longer melodies with time pressure
- Special feedback patterns
- Unique rewards upon completion

## Tournaments

Weekly competitive events with global leaderboards.

### Structure
- New tournament every Monday
- 7-day duration
- Unlimited attempts to improve score
- Categories: Speed, Accuracy, Streak

### Rewards
- Coins and premium currency
- Exclusive cosmetics
- Achievement badges

## Learning Mode

AI-powered educational experience to improve musical ear training.

### Curriculum Types
1. **Interval Training**: Learn to identify musical intervals
2. **Scale Recognition**: Practice identifying major/minor scales
3. **Rhythm Patterns**: Master common rhythm patterns
4. **Melody Memory**: Improve musical memory capacity

### AI Features
- Adaptive difficulty based on performance
- Personalized lesson recommendations
- Progress tracking and analytics
- Spaced repetition for optimal learning

## Melody Quest (Campaign)

Story-driven adventure through musical history.

### Chapters
1. Classical Era
2. Jazz Age
3. Rock Revolution
4. Electronic Age
5. Modern Hits

### Features
- Progressive difficulty curve
- Unlockable content
- Character progression
- Collectible achievements

## Zen Mode (Wellness)

Relaxed puzzle experience focused on mindfulness.

### Characteristics
- No time pressure
- Calming visual themes
- Ambient soundscapes
- Unlimited hints
- No failure state

### Benefits
- Stress relief
- Casual play sessions
- Accessibility-focused design

## Duels

Real-time multiplayer battles.

### Mechanics
- Matchmaking by skill level
- Simultaneous puzzle solving
- First to solve wins
- Ranking system

## Accessibility

All game modes support:
- High contrast mode
- Screen reader compatibility
- Adjustable text sizes
- Color-blind friendly palettes
- Reduced motion options
- Haptic feedback customization
